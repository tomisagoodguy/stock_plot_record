# 第二張圖:

股價波動率大於6%時，畫出停損線

1. 波動率突增（>20%）
   * 表示市場情緒突然改變
   * 可能預示價格大幅波動
   * 建議提高警覺，考慮降低部位
2. 波動率開始上升（上升>10%）
   * 從低波動環境轉為波動加劇
   * 可能是趨勢改變的前兆
   * 建議審視持倉，準備應對策略
3. 異常低波動（低於60日均值的50%）
   * 市場可能過度平靜
   * 常在大行情前出現
   * 建議密切關注，準備應對突發變化
